## Prerequisites

- [Expo](https://docs.expo.io/)
- [Yarn](https://classic.yarnpkg.com/pt-BR/)

## Installing

```bash

  # Clone repository
  $ git clone https://github.com/pereiradilson/react-native-with-expo.git

  # Enter folder
  $ cd react-native-with-expo

  # Install dependences
  $ yarn

  #Execute in iOS
  $ yarn ios

  #Execute in Android
  $ yarn android

```
